# Vēsma Dūmkalna — CV / Portfolio website

Statiska vienas lapas CV mājaslapa (LV/EN) ar atsevišķu izdrukai/PDF gatavu CV lapu.

## Faili
- `index.html` — galvenā mājaslapa
- `CV.html` — izdrukai optimizēta CV lapa (poga "Saglabāt PDF")
- `assets/` — attēli

## Kā publicēt GitHub Pages
1. Izveido jaunu repozitoriju (piem. `cv`) un augšupielādē visus šī foldera failus.
2. Repozitorijā: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, izvēlies `main` / `root`.
3. Pēc dažām minūtēm lapa būs pieejama: `https://<tavs-lietotajvards>.github.io/cv/`
4. Šo saiti vari pievienot savam LinkedIn profilam.
